# Planning Tools

## Beads (Git-backed Task Tracking)

- Distributed, git-backed graph issue tracker for AI agents
- Dependency-aware task graph with semantic "memory decay" (compaction)
- Hash-based IDs prevent merge collisions
- Works with Claude Code natively
- `bd init` in project, `bd ready` to see actionable tasks
- https://github.com/steveyegge/beads

## Linear (Team Projects)

- Used for Black Atom Industries issue tracking
- Best for team collaboration with sprints and roadmaps
- Integrates with GitHub PRs

## Markdown Plans (Solo/Small Projects)

- `docs/plans/YYYY-MM-DD-<topic>.md` convention
- Good for solo work or small scope
- Version controlled, reviewable
- Used in this repo (dots)
