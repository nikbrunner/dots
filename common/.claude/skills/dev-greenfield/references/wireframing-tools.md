# Wireframing Tools

## OpenPencil (Preferred for AI-assisted design)

- Open-source Figma alternative, AI-native
- Built-in chat with tool use (BYOK -- bring your own API key)
- Opens .fig files natively, copy/paste between Figma
- Fully local, no subscription, ~5MB desktop app
- MIT licensed
- https://openpencil.dev/

## Excalidraw (Quick Sketches)

- Hand-drawn aesthetic, great for quick wireframes
- Collaborative, browser-based
- Good for brainstorming sessions
- https://excalidraw.com/

## Figma (Real Designs)

- Industry standard for polished designs
- Use when pixel-perfect mockups are needed
- AI-assisted design emerging (Figma AI, plugins)

## ASCII Layouts

Acceptable for very quick component layout sketches in chat, but prefer visual tools when possible.
