# Wireframing Tools

## OpenPencil (Primary — MCP integrated)

- Open-source Figma alternative, MIT licensed
- **90 MCP tools** via `openpencil-mcp` — Claude can create/edit/export designs directly
- Local `.fig` files (Figma-compatible), PNG/SVG export
- Built-in AI chat (BYOK), also scriptable via CLI
- Full details in the `open-pencil` skill (official, installed via `npx skills add`)
- https://openpencil.dev/

## Excalidraw (Quick Sketches)

- Hand-drawn aesthetic, great for quick wireframes
- Claude can write `.excalidraw` JSON files directly (no MCP needed)
- Viewable in Obsidian (Excalidraw plugin) or excalidraw.com
- Good for brainstorming when a full design tool is overkill
- https://excalidraw.com/

## Figma (Real Designs)

- Industry standard for polished designs
- Use when pixel-perfect mockups are needed
- OpenPencil opens `.fig` files natively — bridge between wireframe and Figma

## ASCII Layouts

Acceptable for very quick component layout sketches in chat, but prefer visual tools when possible.
