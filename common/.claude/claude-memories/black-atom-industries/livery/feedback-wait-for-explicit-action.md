---
name: Wait for explicit action requests
description: Don't act on confirmational/follow-up questions — only edit/install when explicitly asked
type: feedback
---

Do not treat follow-up questions as confirmation to proceed with changes. Questions like "Is this hooked into the dev server?" or "Yes that sounds good" are informational — not instructions to start editing.

**Why:** Nik asked about how a plugin works, and I immediately started installing it. This was premature and unwanted.

**How to apply:** Only make changes (edits, installs, file ops) when Nik explicitly asks to do so (e.g., "add it", "make the change", "go ahead and install"). Confirmational or curious questions are not action requests.
