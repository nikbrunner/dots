# Raycast Script Commands

## Setup

Raycast doesn't auto-detect script directories. You need to register this folder manually once per machine:

1. Open Raycast Settings (`Cmd + ,`)
2. Go to **Extensions** → **Script Commands**
3. Click **"Add Directories"**
4. Select this folder: `~/repos/nikbrunner/dots/macos/raycast/scripts`
5. Search for "Reload Script Directories" in Raycast and run it

## Scripts

| Script                  | Description                                                                                               |
| ----------------------- | --------------------------------------------------------------------------------------------------------- |
| `linear-to-obsidian.sh` | Converts a Linear issue URL from clipboard to Obsidian-friendly markdown link with `linear://` URL scheme |
